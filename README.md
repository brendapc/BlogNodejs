# BlogNodeJs

sistema que permite que apenas administradores criem, deletem, atualizem os dados de uma pagina (blog).
<hr>
Principais tecnologias usadas:
<ul>
  <li>NodeJS</li>
    <ul>
     <li>Express.js</li>
     <li>Express-Handlebars</li>  
     <li>Mongoose</li>  
    </ul>
 <li>MongoDB</li>
 <li>Bootstrap</li>
 <li>Passport</li>
 <li>Bcrypt</li>
</ul>
